# Professional Features Guide

Profesjonelle funksjoner er laget for strukturert oppfølging, rapportering og manuell gjennomgang. De er ikke en medisinsk vurdering alene.

## Funksjoner

- Klinisk oversikt.
- Coach-oversikt.
- Rapporter.
- Manuell justering.
- Profesjonelle notater.
- Audit trail.
- Case review.

## Personvern

Profesjonelle notater kan inneholde sensitiv fritekst. De er ekskludert fra research export og support package som standard.

## Hva testere bør sjekke

- At dashboard åpner uten exceptions.
- At notater kan lagres og leses tilbake.
- At audit trail registrerer handlinger.
- At rapporter kan eksporteres.
- At profesjonelle notater ikke lekker inn i support/research som standard.

## Begrensning

Profesjonelle funksjoner støtter arbeid og dokumentasjon, men erstatter ikke kvalifisert klinisk vurdering.
