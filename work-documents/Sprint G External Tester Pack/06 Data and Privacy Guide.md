# Data & Privacy Guide

## Grunnprinsipp

FemVoice er lokal-first. Ingen cloud upload eller skjult telemetry skal være aktiv som standard.

## Data som lagres lokalt

FemVoice kan lagre:

- Innstillinger.
- Økthistorikk.
- Analytics/progresjon.
- Målprofiler.
- Mikrofonkalibrering.
- Rapportmetadata.
- Profesjonelle notater hvis du bruker profesjonelle funksjoner.

## Diagnostikk

Diagnostikk og support package export er bruker-kontrollert. Det skal ikke eksporteres automatisk uten at du selv starter det.

## Support package

Support package kan inneholde:

- Runtime log.
- Evidence JSON.
- Verification report.
- Audio diagnostic report.
- Errors-only.
- Screenshot checklist.
- Session summary.
- App/system/settings summary.

Ekskludert som standard:

- Profesjonelle fritekstnotater.
- Personlige notater.
- Identifiserbare research exports.
- Secrets eller tokens.
- Privat råtekst.

## Research export

Research export skal anonymiseres som standard. Bruker-id, mikrofonnavn, tidsdetaljer og fritekst skal fjernes eller generaliseres der det er relevant.

## Profesjonelle notater

Profesjonelle notater kan inneholde sensitiv fritekst. De skal ikke inkluderes i research export eller support package som standard.

## Hva du bør unngå å sende inn

Ikke send:

- Private notater.
- Personopplysninger.
- Fullt navn/adresse.
- Sensitive helseopplysninger.

med mindre det er helt nødvendig for feilsøking og du eksplisitt ønsker å dele det.
