# Troubleshooting Guide

## Mikrofon ikke tilgjengelig

Prøv:

1. Kontroller at mikrofonen er koblet til.
2. Sjekk Windows personverninnstillinger for mikrofon.
3. Lukk andre programmer som bruker mikrofonen.
4. Velg riktig mikrofon i FemVoice.
5. Start appen på nytt.

Forventet appadferd:

- Appen skal ikke krasje.
- Den skal vise trygg melding.
- Du skal kunne prøve igjen eller velge device.

## Rapporteksport feiler

Prøv:

1. Velg en mappe du har skrivetilgang til.
2. Lukk PDF-filen hvis den allerede er åpen.
3. Prøv eksport på nytt.
4. Lag support package hvis feilen gjentar seg.

## Support package export

Support package lages fra appen når du selv velger det. Den skal ikke lastes opp automatisk.

Hvis eksport feiler:

- Velg en annen mappe.
- Kontroller diskplass.
- Start appen på nytt og prøv igjen.

## Backup/restore

Backup:

- Bruk lokal backup før større testing eller endringer.
- Lagre backup på et sted du finner igjen.

Restore:

- Restore over eksisterende data skal be om bekreftelse.
- Ikke avbryt appen mens restore pågår.
- Hvis restore feiler, skal eksisterende data være beskyttet.

## Tom analytics/session history

Dette kan være normalt hvis:

- Du nettopp startet på ny installasjon.
- Ingen økter er fullført.
- Data ble gjenopprettet fra backup uten historikk.

## Hvordan finne diagnostikk/support package

Diagnostikk ligger lokalt under FemVoice Studio sin LocalAppData-mappe. Support package lagres der du velger ved eksport, eller i appens standard supportpakke-mappe hvis det brukes.

## Appen starter ikke

Prøv:

1. Start PC-en på nytt.
2. Kontroller at installasjonsmappen finnes.
3. Start appen fra pakket/installert plassering, ikke fra midlertidig testmappe.
4. Lagre eventuell feilmelding eller skjermbilde.
5. Send support package hvis du kommer inn i appen etterpå.
