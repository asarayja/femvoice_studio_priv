# FemVoice Studio - Release Notes for kontrollert ekstern testing

Status: Release candidate for kontrollert ekstern testing  
Målgruppe: eksterne testere, pilotbrukere og release candidate testere

## Hva som er klart for testing

FemVoice Studio inneholder nå:

- Øvelsesguide med sanntidsveiledning.
- Audio-, pitch- og resonansbasert biofeedback.
- SmartCoach og progresjonsvisning.
- Voice Health- og pauseveiledning.
- Mikrofonkalibrering per device.
- Rapporter, eksport og profesjonelle arbeidsflater.
- Backup/restore.
- Lokal support package export.
- Privacy-first research export med anonymisering som standard.

## Viktig ramme

FemVoice er et trenings- og feedbackverktøy. Det gir støttende veiledning for stemmeøving, men er ikke en medisinsk vurdering, diagnose eller erstatning for fagperson.
