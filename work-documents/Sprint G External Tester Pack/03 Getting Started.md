# Getting Started for eksterne testere

## Før du starter

Du trenger:

- Windows-PC.
- Mikrofon, helst samme mikrofon du vanligvis bruker.
- Rolig testmiljø.
- FemVoice Studio installert eller pakket for testing.

## Første oppstart

1. Start FemVoice Studio.
2. Fullfør første oppsett hvis det vises.
3. Velg språk og tema.
4. Kontroller at mikrofonen vises i innstillinger eller mikrofonoppsett.
5. Kjør mikrofonkalibrering hvis appen ber om det.

## Første testøkt

1. Åpne hovedsiden.
2. Start en enkel øvelse.
3. Bekreft at timeren går.
4. Bekreft at pitch/resonans/feedback oppdateres når du lager lyd.
5. Stopp økten.
6. Sjekk at resultat/progresjon blir lagret.

## Hvor lagres data?

FemVoice lagrer data lokalt på PC-en. Diagnostikk lagres under LocalAppData for FemVoice Studio, mens brukerrettede rapporter og eksportfiler lagres på valgt eksportplassering.

## Hvis noe går galt

Lag en support package og fyll ut feedback template. Ikke lim inn private notater eller sensitiv fritekst med mindre du eksplisitt ønsker at utvikler skal se det.
