# External Tester Checklist

Bruk denne listen ved kontrollert ekstern testing.

## Før test

- [ ] Noter appversjon.
- [ ] Noter Windows-versjon.
- [ ] Noter mikrofon/device.
- [ ] Bekreft at du tester riktig release candidate.
- [ ] Lag backup hvis du har eksisterende data.

## Første oppstart

- [ ] App starter uten krasj.
- [ ] Første oppsett/onboarding vises hvis relevant.
- [ ] Innstillinger kan lagres.
- [ ] Diagnostikk-/rapportmapper fungerer.
- [ ] Ingen Debug/temp-avhengighet merkes.

## Øvelse

- [ ] Start øvelse.
- [ ] Timer går.
- [ ] Audio/pitch/resonans oppdateres.
- [ ] Feedback vises.
- [ ] Stopp øvelse.
- [ ] Resultat lagres.

## Rapporter og eksport

- [ ] Rapport genereres.
- [ ] PDF åpner.
- [ ] CSV/JSON kan leses.
- [ ] Norske timeline labels er riktige.
- [ ] Ingen `Restitusjonskostn/ad`.

## Privacy/support

- [ ] Support package kan lages.
- [ ] Sensitive notater er ikke med som standard.
- [ ] Research export er anonymisert.

## Etter test

- [ ] Fyll ut feedback template.
- [ ] Legg ved skjermbilde hvis relevant.
- [ ] Legg ved support package hvis feil oppstod.
