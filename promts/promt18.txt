## Clinical Foundation

Resonance feminization depends on vocal tract shaping, not pitch. The system must:

- Focus on formant frequency shifts (F1, F2, F3) that indicate forward tongue positioning and oral resonance
- Detect spectral brightness patterns associated with feminine resonance
- Score vocal tract configuration independently—even feminine pitch with masculine formant structures must score low
- Correlate strongly with perceived vocal femininity in clinical voice training contexts

## Core Acoustic Features

Implement real-time extraction of these features from NAudio audio buffers:

**Formant Frequencies (F1, F2, F3)**
- Use FFT + LPC or robust spectral peak tracking compatible with NAudio's processing pipeline
- Track stable formant peaks across consecutive frames
- Handle voice breaks and momentary instability gracefully

**Formant Spacing Metrics**
- F2 - F1 distance (indicates tongue advancement)
- F3 - F2 distance (indicates oral cavity resonance)

**Spectral Centroid**
- Calculate within the vocal band (200-4000 Hz)
- Measure perceived brightness of resonance

**Temporal Stability**
- Track formant consistency across frames
- Penalize unstable resonance patterns that would cause noisy feedback

## ResonanceScore Algorithm

Design a normalized 0.0–1.0 scoring system that combines weighted components:

```
ResonanceScore = 
  0.40 * FormantShiftScore +
  0.25 * SpacingScore +
  0.25 * BrightnessScore +
  0.10 * StabilityScore
```

**For each component:**

1. Define clinically realistic bounds (e.g., F2 target zone: 2200-2800 Hz for feminine resonance)
2. Normalize extracted values to 0.0–1.0 using those bounds
3. Make weights configurable via constructor or properties

**Scoring behavior requirements:**
- Reward forward tongue positioning (higher F2)
- Reward higher oral resonance (increased F3, wider F2-F1 spacing)
- Penalize dark/throaty resonance (low centroid, narrow formant spacing)
- Remain completely independent of pitch/F0
